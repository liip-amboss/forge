# Modal

## Usage
```vue
<forge-modal>
  <p slot="header">Title</p>
  <div>...</div>
  <p slot="footer">Button</p>
</forge-modal>
```

## Props
| Prop name          | Type          | Default  |
|:-------------------|:--------------|:---------|
| is-open (required) | Boolean       | false    |
| class-modal        | String        | ""       |
| class-backdrop     | String        | ""       |
| class-container    | String        | ""       |
| class-header       | String        | ""       |
| class-btn-close    | String        | ""       |
| class-body         | String        | ""       |
| class-footer       | String        | ""       |
