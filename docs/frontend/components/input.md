# Input

## Usage
```vue
<forge-input value="Mike" label="First name" />
```

## Props
| Prop name          | Type          | Default  | Required |
|:-------------------|:--------------|:---------|----------|
| value              | String        |          | yes      |
| label              | String        |          | yes      |
| is-block           | Boolean       | false    |          |
| type               | String        | "text"   |          |
| disabled           | Boolean       | false    |          |
| placeholder        | String        | ""       |          |
| class-input        | String        | ""       |          |
| class-label        | String        | ""       |          |
| max-length         | Number        | null     |          |
